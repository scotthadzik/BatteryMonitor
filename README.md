# Setup Instructions
Runs on Raspian without Desktop

## External Modules
Run this before running program
- sudo apt install python3-pip
- sudo pip3 install guizero
- sudo apt update
- sudo apt upgrade
- modify /boot/config.txt
	- dtoverlay=w1-gpio
- sudo reboot
